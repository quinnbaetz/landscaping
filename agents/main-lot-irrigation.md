# Main lot — Phase 3: Irrigation & Well Pump

## Source

Existing well on the property. Keep the casing/bore as-is. **Replace the pump.** Drip system downstream.

## Components

| Component | Spec |
|---|---|
| Pump | ½–¾ HP, 10–15 gpm at 40–60 psi, stainless wet end. **Submersible** if static water level > 25 ft, **shallow-well jet pump** otherwise |
| Pressure tank | 20–30 gal bladder, switch 40 psi cut-in / 60 psi cut-out |
| Sediment filter | Inline 5 micron cartridge after the pressure tank |
| Iron filter | Optional — add only if water test shows > 0.3 ppm Fe (else drip emitters clog) |
| Backflow preventer | Single-check minimum; reduced-pressure assembly if locally required |
| Mainline | ¾ in or 1 in poly pipe, buried 12 in under driveway transitions |
| Zone manifold | 4–6 manual ball valves at single hub. Battery timer per zone optional later |
| Sub-mainlines | ½ in poly to each zone |
| Drip | ¼ in line with pressure-comp emitters (1–2 gph) |

## Zones (5 total)

| # | Zone | Coverage |
|---|---|---|
| 1 | North band trees | T1, T2, T14, T17 |
| 2 | West band trees | T3, T4, T5, T6, T7, T8 |
| 3 | East band trees | T9, T10, T11, T12, T13 |
| 4 | South corners + east wall trellis | T15, T16, blackberry, grape, fan-trained fig |
| 5 | Lawn | Pop-up spray or rotor heads on separate valve |

## Emitter counts per plant

- Fruit tree: 4–6 emitters at drip line
- Bridge shrub: 2 emitters
- Guild perennials: share a soaker loop in each tree's mulch ring

## Run times (summer, established trees)

- Tree zones: 30–45 min, 2–3× per week
- Lawn: separate schedule

## Build order

1. Test well water (Fe, hardness, bacteria), static level, recovery rate
2. Install pump + pressure tank + filter + backflow at pump house
3. Trench/lay mainline to lot; pressure-test before burying
4. Build zone manifold at hub
5. Run ½ in submains across each planting zone **before topsoil cap goes down** (pin where tree positions will be)
6. After Phase 2 planting: attach ¼ in drip + emitters at each tree
7. Full-cycle commissioning: check every emitter, look for leaks, adjust run times

## Scope intentionally excluded (can layer in later)

- Rainwater harvesting
- Greywater (Laundry-to-Landscape)
- Smart ET-based controller
